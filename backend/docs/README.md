# Documentation API SkillHub

La documentation officielle de l'API est définie dans `openapi.yaml`.

## Choix retenu

- **OpenAPI 3.0** comme source de vérité
- **Swagger UI** possible plus tard pour l'affichage interactif

## Fichiers

- `docs/openapi.yaml` : spécification complète de l'API

## Utilisation

Tu peux ouvrir ce fichier dans :
- Swagger Editor
- Swagger UI
- Redoc
- n'importe quel visualiseur OpenAPI compatible YAML

## Périmètre couvert

- Authentification JWT
- Formations
- Modules
- Inscriptions apprenant
